This repository have include the pevious and latest versions of different calculators used in our Web Application, that is, zer0carb0n.me.The pages with extension .html are the latest pages that are being used in our website. These pages fetch data from the database and perform input validation using AJAX, JavaScript and Jquery. Further, test.php and test2.php are the two endpoints for AJAX POST request. Test.php servers the AJAX request to make the dropdown list dynamic and Test2.php serves the AJAX request to fetch the Carbon emissions of particular car selected by the user.
